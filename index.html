몽글 개인정보처리방침

몽글은 사용자의 개인정보 보호를 최우선으로 생각하며, 다음과 같은 원칙에 따라 개인정보를 처리합니다.

1. 수집하는 개인정보 항목

몽글은 사용자의 개인정보를 서버에 저장하지 않으며, 다음의 정보를 로컬 또는 외부 API와의 통신 과정에서만 일시적으로 처리합니다.

  • AI 채팅 요청 내용 (DeepSeek API로 전달되며, 앱 자체 서버에는 저장되지 않음)
  • Firebase 프로젝트를 통한 기기 인증 정보 (식별 불가능한 형태)
  • AdMob 광고 노출을 위한 광고 ID (Google에서 자동 처리)

2. 개인정보 수집 및 이용 목적

몽글은 아래의 목적에 한하여 비식별 정보를 수집하거나 처리할 수 있습니다:

  • AI 챗봇 응답 제공을 위한 DeepSeek API 요청 처리
  • 앱 보안을 위한 Firebase App Check 사용
  • 앱 내 광고 제공 (Google AdMob)

3. 개인정보 보관 및 처리 방식

몽글은 사용자로부터 받은 어떤 데이터도 자체 서버에 저장하지 않습니다. 모든 대화는 기기 내에서만 처리되며, AI 응답을 위해 DeepSeek API로 전달되는 대화 내용은 일시적으로 처리된 후 삭제됩니다. DeepSeek의 데이터 처리 정책은 해당 서비스의 개인정보처리방침을 참조하시기 바랍니다.

4. 제3자 제공

몽글은 사용자가 입력한 대화 내용을 AI 응답 생성 목적으로 DeepSeek API 서버로 전송합니다.

  • 전송 경로: Vercel(프록시 서버) → DeepSeek API 서버
  • 수집 항목: 사용자의 채팅 입력 텍스트
  • 이용 목적: AI 챗봇 응답 생성
  • 보관 기간: 서버에 저장하지 않으며, 요청 처리 이후 즉시 폐기

5. 이용자의 권리

몽글은 사용자의 개인정보를 보유하지 않기 때문에, 별도의 열람·수정·삭제 요청이 필요하지 않습니다.
